# Trabajo Práctico - Inferencia Bayesiana  
## Caso B: Banco Ánfora

Repositorio del trabajo práctico del Taller Integrador de Inferencia Bayesiana, Maestría en Minería de Datos, FCEN-UBA.

El caso propone un reanálisis bayesiano de un sistema de detección de fraude del Banco Ánfora, luego de un cambio de distribución producido por una intervención externa: una campaña masiva de phishing por SMS.

El objetivo general es evaluar el desempeño de cinco modelos candidatos de detección de fraude y recomendar qué modelo debería desplegarse a partir de julio de 2025, incorporando incertidumbre mediante inferencia bayesiana.
